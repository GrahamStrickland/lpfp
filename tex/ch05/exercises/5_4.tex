We have the function \verb|range| defined in \path{src/Ch05/Range.hs}.

\verb|range| returns a list containing all the integers between the argument 
(inclusive) and 0 in increasing order, i.e, $\verb|range|(2) = 0, 1, 2$, 
$\verb|range|(-4) = -4, -3, \ldots, 0$, and $\verb|range|(0) = 0$.\par

\qquad We demonstrate as follows:
\begin{verbatim}
ghci> range (-4)
[-4,-3,-2,-1,0]
ghci> range 2
[0,1,2]
ghci> range 0
[0]
\end{verbatim}
